# ISData Consulting

Site for isdataconsulting.com.

## Requirements

- Docker

## Setup

```bash
cp .env.example .env
```

## Development

```bash
npm run dev
```

## Quality

```bash
npm run lint
npm run type-check
npm run test
npm run test:coverage
```

## Build

```bash
npm run build
npm run preview
```

## Deployment

Netlify builds with `npm run build` and publishes `dist/`.
